## Deliberately not adopted from claude-hud

- **`(1h 30m / 5h)` elapsed-style usage display** тАФ super-status's countdown + reset clock (`Reset: 2h30m [12:40]`) already carries more information; a `timeFormat` config toggle can ride on I5 later if wanted.
- **Node.js/Bun runtime** тАФ staying a dependency-light bash script is a feature of this repo, not a gap.
- **`--extra-cmd` arbitrary-command hook** тАФ a gated escape hatch for injecting external command output; adds an attack surface (claude-hud itself gates it behind an env var) with no current need here.
